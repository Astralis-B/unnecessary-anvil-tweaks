# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-21
### Initial Release
- **Added**: 5 animation styles (Slide, Fade, Slide Up, Typewriter, Slide+Fade)
- **Added**: Text alignment options (Left, Center, Right)
- **Added**: Shake effect for "Too Expensive" messages
- **Added**: Animation speed control (0.1x to 3.0x)
- **Added**: ModMenu integration for in-game configuration
- **Added**: Multi-language support (English and Spanish)
- **Added**: Clean UI without background boxes
- **Technical**: Client-side only implementation
- **Technical**: Built for Minecraft 1.21.11 with Fabric
